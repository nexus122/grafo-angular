Instrucciones para utilizar el grafo

Para utilizar el grafo, siga las siguientes instrucciones:

  • Haga doble clic en un nodo para abrir el modal.
  • En el modal, puede cambiar el nombre del nodo.
  • Puede agregar una descripción al nodo.
  • Guarde o borre el nodo según sea necesario.
  • Para unir nodos, simplemente acérquelos.
